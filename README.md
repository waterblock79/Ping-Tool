# Ping-Tool
Is a ping tool bulid in cpp.

![image.png](https://i.loli.net/2020/04/11/XsgVWDBptzQMf24.png)

The ping windows will automatic minimization,and it won't effect your work

You can set up these ping settings:

* Ping bag size
* Ping website or IP
* Ping **threads**
